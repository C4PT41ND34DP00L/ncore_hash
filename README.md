# ncore_hash
python multi-core rainbow table generator

**Supported Hash types:**
* MD5
* SHA1
* SHA224
* SHA256
* SHA384
* SHA512
* RipeMD160

**Example:**
`$ python ncore_hash.py -d ~/Desktop -l 2-5 --hash MD5 --full_ascii -s sa1ty`

**Tested on:**
OSX 10.11.3 Mac Mini 2014
